# My Capstone Portfolio Project

A professional developer portfolio designed to communicate technical proficiency, design quality, and project achievements. The site combines a clear information hierarchy with responsive presentation and interactive features to deliver a refined user experience.

## Tech Stack

This project leverages industry-standard web technologies to achieve accessibility, responsiveness, and modern interface behavior.

- HTML5
  - Semantic markup for clear structure and improved accessibility
  - Organized content hierarchy for searchability and maintainability
- CSS3
  - Responsive layout techniques for consistent presentation across devices
  - Visual styling and transitions to enhance user experience
- JavaScript
  - Interactive behavior for navigation, content display, and dynamic updates
  - Performance-conscious scripting for smooth operation and usability
